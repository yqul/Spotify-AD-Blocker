# spotify-adblock-bilal

blocks audio ads + banner ads + upgrade popups.  
no premium needed. just runs and works.

## what it does
- kills spotify
- blocks ad servers in hosts
- removes top banner (xpui edit)
- deletes ad cache
- stops auto updates

## how to use
1. close spotify  
2. run install.bat as admin  
3. open spotify  
4. done.

to undo: run uninstall.bat as admin

## tested on
windows 10 / 11  
spotify latest version
